# Online Food Delivery Database System

## Overview

This project is a MySQL-based Online Food Delivery Database Management System. It manages users, restaurants, menu items, orders, and order tracking efficiently using relational database concepts.

## Technologies Used

* MySQL
* SQL

## Features

* User Management
* Restaurant Management
* Menu Management
* Order Processing
* Order Tracking
* Revenue Analysis

## Database Tables

1. Users
2. Restaurants
3. Menu
4. Orders
5. Order_Items
6. Order_Tracking

## Key SQL Operations

* Create Database and Tables
* Primary & Foreign Key Constraints
* Joins
* Aggregate Functions
* Order Tracking Queries
* Revenue Calculation Queries

## How to Run

1. Open MySQL Workbench.
2. Create a new SQL file.
3. Copy and paste the provided SQL script.
4. Execute the script.
5. Run the sample queries to test the database.

## Project Outcome

* Designed a normalized food delivery database.
* Implemented relationships using foreign keys.
* Performed data retrieval and analysis using SQL queries.
